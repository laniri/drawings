# GET /api/v1/drawings/upload/progress/{upload_id}

## Summary
Get Upload Progress

## Description
Get upload progress for large file uploads.

## Parameters
- **upload_id** (path): No description

## Responses
- **200**: Successful Response
- **422**: Validation Error

## Example
```http
GET /api/v1/drawings/upload/progress/{upload_id}
```
